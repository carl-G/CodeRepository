## Git详解

### git简介

git是目前最流行的版本管理工具，学会git几乎成了开发者的必备技能。
git有很多优势，其中之一就是远程操作非常简便，一下罗列最常用的5个git命令

+ git clone
+ git remote
+ git fetch
+ git pull
+ git push
